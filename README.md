# CocoyMango

Las Aventuras de Coco y Mango - An interactive Spanish-language children's story reading platform.

## Description

A React-based web application for reading children's stories featuring characters Coco and Mango. The platform offers multiple font sizes, font families, dark mode, and tracks reading statistics.

## Key Features

- 12 illustrated children's stories in Spanish
- Multiple font size options (5 sizes)
- Multiple font family options (sans, serif, mono, comic, rounded)
- Dark mode support
- Reading progress tracking
- Story navigation menu
- Mobile-responsive design

## Tech Stack

- React 18 (via CDN)
- Tailwind CSS (via CDN)
- Babel (for JSX transformation)
- JSON story data format

## How to Run

Open `index.html` in a web browser. The stories are loaded from the `historias/` directory as JSON files.

## Project Structure

```
CocoyMango/
├── index.html           # Main React application
├── historias/           # Story JSON files (historia01.json - historia12.json)
├── imagenes/            # Story illustrations
└── all-stories.txt      # Story text compilation
```
